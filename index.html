Google Drive
Importa manuales automáticamente desde una carpeta de Drive
⚙ Configuración
Obtener en console.cloud.google.com → APIs → Drive API → Credenciales
Es el trozo largo en la URL de la carpeta: drive.google.com/drive/folders/ESTE_ID
🌐 Sincronización global (todos los dispositivos)
Genera el archivo config.js y súbelo a GitHub junto con el index.html. Cualquier dispositivo que abra la URL sincronizará automáticamente sin configuración extra.
Súbelo a GitHub junto al index.html
🔄 Sincronización
📱 Acceso desde otros dispositivos
Genera un enlace que configura Google Drive automáticamente en cualquier dispositivo (móvil, tablet…) cuando se abre.
⚠ Este enlace contiene tu API Key. Compártelo solo con personas de confianza.
❓ Cómo configurarlo
  1. Ve a Google Drive y crea una carpeta llamada Manuales
  2. Haz clic derecho en la carpeta → Compartir → Cualquiera con el enlace puede ver
  3. Copia el ID de la URL y pégalo arriba
  4. Ve a console.cloud.google.com
  5. Crea un proyecto → Habilitar APIs → busca Google Drive API → Habilitar
  6. Ve a Credenciales → Crear credencial → Clave de API
  7. Copia la clave y pégala arriba
  8. Sube tus PDFs y documentos a la carpeta de Drive
  9. Pulsa Sincronizar ahora
Los archivos de Drive se previsulaizan en el visor sin necesidad de descargarlos. El almacenamiento es ilimitado (usa la cuota de tu Google Drive).
Sincronización entre dispositivos (Firebase)
Conecta Firebase para sincronizar todos los datos (instalaciones, historial, trabajos pendientes, fotos, comentarios) en tiempo real entre todos los dispositivos. Gratuito hasta 1GB.
Configuración (5 min):
1. Ve a console.firebase.google.com → Crear proyecto
2. Firestore Database → Crear base de datos → Modo producción
3. Configuración del proyecto → Tus apps → Web → Registrar app
4. Copia el objeto firebaseConfig y pégalo aquí
Usa el mismo ID en todos los dispositivos para que compartan los datos.
